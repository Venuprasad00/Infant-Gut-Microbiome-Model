# Infant-Gut-Microbiome-Model
Code accompanying the manuscript on mechanistic modeling of infant gut microbiome development, microbiome restoration after caesarean delivery, and evaluation of nutritional interventions

## Repository Contents 

- **ODEModelFile.jl** – Julia implementation of the mechanistic ordinary differential equation (ODE) model describing microbial population dynamics, substrate utilization, microbial competition, oxygen depletion, and feeding transitions. - **MLE_Parameter_Estimates_MATLAB.csv** – Final parameter estimates and 95% confidence intervals obtained from MATLAB-based maximum likelihood estimation. ## Model Features - Vaginal and caesarean delivery scenarios - Breastfeeding, prebiotic, and synbiotic interventions - Dynamic HMO, GOS, FOS, dietary fiber, and oxygen utilization - Weaning transition - Microbial competition among four functional groups ## Citation If you use this repository, please cite the accompanying manuscript.
